## 9. CSS Design Approach

The website uses a dedicated style.css file that applies consistent design across all pages. Key techniques used:

- **Flexbox Layouts:** Used for aligning images/icons next to headings and creating responsive row/column layouts.
- **Grid Structure:** Applied to organize service offerings and form sections neatly.
- **Typography Hierarchy:** Montserrat for headings and Open Sans for body text to ensure readability and professionalism.
- **Responsive Design:** Media queries adjust layouts for mobile, tablet, and desktop.
- **Color Scheme Consistency:** Red for action elements, black for text emphasis, white/grey for backgrounds.
- **Button Styling:** Bold CTAs with hover effects for clear interactivity.
- **Form Styling:** Input fields and textareas styled with padding, borders, and consistent spacing for usability.
- **Image/Icon Sizing:** Controlled with max-width and flex alignment to ensure uniform presentation.
