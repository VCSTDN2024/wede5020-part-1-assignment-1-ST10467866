##  1. Organisational Overview

**Company Name:** Lady Leah’s Transport (LLT)  
**Established:** 2017  
**Location:** Johannesburg, South Africa  

Lady Leah’s Transport (LLT) is a trusted logistics and courier company operating across Johannesburg and Pretoria. Founded in 2017, LLT provides **fast**, **reliable**, and **cost-effective** transportation services for individuals and businesses.  

The company’s focus on **professionalism**, **punctuality**, and **customer satisfaction** has built strong relationships with SMEs, corporates, and private clients.

**Mission Statement:**  
> “To provide affordable, secure, and timely transport solutions that connect people and businesses with efficiency and trust.”

**Vision Statement:**  
> “To become Gauteng’s leading transport and logistics company, recognized for innovation, customer satisfaction, and reliability.”

**Target Audience:**  
- Small and medium-sized enterprises (SMEs)  
- Corporate delivery clients  
- Individual courier customers  
- NGOs and community delivery partners

### **Goals**

##  3. Website Structure and Features

### **Home Page**
- Hero banner with slogan and call-to-action buttons  
- Overview of services and delivery benefits  
- Customer testimonials and trust badges  
- Links to booking and freight analysis  

### **About Page**
- Detailed company background, mission, and values  
- History of growth and team overview  
- Focus on professionalism and customer trust  
- Corporate social responsibility (community outreach)  

### **Services Page**
- Full breakdown of services: same-day, overnight, bulk, and corporate logistics  
- **Accordion sections** implemented with JavaScript for collapsible service info  
- FAQ section for customer clarity  
- Quick “Request a Quote” link  

### **Contact Page**
- Interactive contact form with JavaScript validation  
- Embedded **Google Map** for office location  
- Clickable contact details (email, WhatsApp, phone)  
- Fully responsive layout  

### **Free Freight Analysis**
- Quote request form for businesses and bulk clients  
- File upload option for detailed requests  
- Auto confirmation message with validation  
- Delivery savings insights  

### **Courier Locator**
- Live map integration (Google Maps iframe)  
- Delivery tracking input field  
- Estimated delivery time display

##  4. Design and User Experience (UX/UI)

### **Color Scheme**
| Color | Code | Purpose |
|-------|-------|----------|
| Red | `#B71C1C` | Represents trust, power, and speed |
| Black | `#212121` | Professionalism and sophistication |
| White | `#FFFFFF` | Clean layout and readability |
| Light Grey | `#F5F5F5` | Visual balance and modern tone |

### **Typography**
- **Headings:** Montserrat (modern, bold, and strong)  
- **Body Text:** Open Sans (simple, readable, and elegant)

### **Layout**
- Consistent navigation and footer  
- Flexbox and Grid for balanced structure  
- Clean sections with strong visual hierarchy  
- Minimal clutter and easy-to-scan design  

### **UX Considerations**
- Mobile-first responsive approach  
- Alt text and ARIA labels for accessibility  
- Fast load times with optimized media  
- Intuitive navigation (3-click rule)

## 5. CSS Implementation

All styles are defined in a single external stylesheet: `styleSheet.css`.

### **Key CSS Features**
- **Flexbox** and **Grid** for structure and alignment  
- **Hover transitions** and **animations** for interactivity  
- **Rounded corners**, **shadows**, and **cards** for modern aesthetics  
- **Accordion styling** with active state indicators  
- **Media queries** for responsiveness on all devices  
- **Typography hierarchy** for readability and emphasis

##  6. JavaScript Functionality

File: `script.js`  

### **Implemented Features**
- Accordion toggles on the Services page  
- Client-side **form validation** for inputs and emails  
- Smooth scroll and fade animations  
- Google Maps integration on Contact and Courier Locator pages  
- Responsive navigation toggle (hamburger menu)  
- Confirmation messages after successful form submissions  

### **Libraries & APIs**
- Vanilla JavaScript (no frameworks)  
- Google Maps Embed API (iframe integration)
